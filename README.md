# 🎓 EduTrack AI

[![Status](https://img.shields.io/badge/Status-Active-brightgreen)](https://github.com/your-repo/edutrack-ai)

**EduTrack AI is a smart academic planner designed to help students organize their academic life, track progress, and manage tasks efficiently.**

This application provides a user-friendly interface to manage subjects, tasks, and monitor academic performance through insightful reports.

---

## ✨ Features

-   **👤 User Authentication:** Secure login and profile management.
-   **📚 Subject Management:** Add, edit, and delete school subjects.
-   **📝 Task Tracking:** Create, update, and manage academic tasks with priorities and due dates.
-   **📊 Progress Reports:** Visualize your academic progress with charts and metrics.
-   **⬇️ Data Export:** Export your data to CSV or JSON for backup or analysis.

---

## 🚀 Getting Started

To run this application locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-repo/edutrack-ai.git
    cd edutrack-ai
    ```

2.  **Install the required Python packages:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Streamlit application:**
    ```bash
    streamlit run HOMEPAGE.py
    ```

The application will open in your default web browser.
